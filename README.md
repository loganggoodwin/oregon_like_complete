## 🚀 Featured Project: Trail Game (Oregon Trail–style) in Python (Pygame)

* Travel, hunt, rest, manage supplies
* Named party members with individual health/status
* Rivers with crossing choices (ford/caulk/ferry)
* Towns/landmarks along the route
* Quantity-based shop
* Save/Load to JSON

### Run it locally

```bash

python oregon_like_complete.py
```

### Controls

* Use on-screen buttons for actions
* In the Shop: click a field, type numbers, Backspace to edit
* **S** = Save, **L** = Load
* Save file: `trail_save.json`
